Usos

La razón principal por la que estoy haciendo esto es porque esperaba poder saber, durante el proceso de personalizar mi interfaz de usuario, cómo hacerlo lo más atractivo y eficiente posible.

Como lo comentó Linus Torvalds:

“Cualquier programa solo es tan bueno como sea útil. En el verdadero open source, tienes el derecho a controlar tu propio destino.”

NOTA: Generalmente se actualiza de forma eventual, por lo que NO puede estar al día esta información.

Lo Básico#

  • Para mi sistema operativo, uso Arch Linux (btw).
  • Para mi window manager, uso Hyprland.
  • Para mi terminal, uso Kitty.
  • Para mi shell, uso Zsh.

Aplicaciones#

  • Para mi launcher, uso Rofi
  • Para mi editor, uso NeoVim
  • Para mi file manager, uso Yazi
  • Para mi terminal multiplexer, uso Tmux

Theme#

  • Para mi theme, generalmente uso Catppuccin
  • Para mi terminal font, uso JetBrains Mono
  • Para mi shell prompt, uso oh-my-posh
